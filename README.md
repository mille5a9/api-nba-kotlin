# Collection of unofficial Kotlin Wrappers for the API-SPORTS APIs

https://api-sports.io/

Implementation Details
-
These wrappers use [Ktor](https://ktor.io/) as an Http Client. Plugins:
- ContentNegotiation (Gson)
- HttpCache
